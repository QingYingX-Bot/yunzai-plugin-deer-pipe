# yunzai-plugin-deer-pipe

一个基于 Yunzai 的🦌管签到插件，支持精力/肾气管理系统，帮助用户进行节制管理。

## 功能特性

- ✅ **签到系统**：支持正常签到、补签、戒🦌
- ✅ **精力管理系统**：无上限精力值，支持透支机制
- ✅ **健康度系统**：肾脏健康度长期指标
- ✅ **惩罚机制**：连续天数、次数、时间等多维度惩罚
- ✅ **恢复机制**：时间恢复、戒🦌恢复、睡眠加速
- ✅ **数据统计**：个人统计、群统计、全局统计、统计排行榜
- ✅ **健康报告**：详细的健康状态和建议

## 安装

```bash
cd plugins
git clone https://github.com/your-repo/yunzai-plugin-deer-pipe
cd ..
pnpm install
```

## 使用

### 基本命令

| 命令 | 功能 | 说明 |
|------|------|------|
| `🦌` / `🦒` / `鹿` / `打胶` | 签到 | 进行当日签到 |
| `补🦌 日期` | 补签 | 补签指定日期（如：补🦌 5） |
| `戒🦌 [日期]` | 戒🦌 | 记录戒🦌，恢复精力 |
| `#鹿鹿查询` / `#鹿鹿查询上月` / `#鹿鹿查询下月` | 查询 | 查看本月或上月签到日历和统计数据 |
| `#鹿鹿统计` | 群统计 | 查看群内统计（仅群聊） |
| `#总鹿鹿统计` / `#鹿鹿总统计` | 全局统计 | 查看全局统计和排行榜 |
| `#鹿鹿体力查询` | 体力查询 | 查看当前精力状态 |
| `#鹿鹿健康报告` | 健康报告 | 查看详细健康报告 |
| `#清空鹿数据` | 清空数据 | 清空所有数据（需确认） |
| `#强制清空鹿数据` | 强制清空 | 强制清空所有数据 |
| `#确认清空鹿数据` | 确认清空 | 确认清空操作 |

### 命令说明

#### 签到相关
- **签到**：发送 `🦌`、`🦒`、`鹿` 或 `打胶` 进行签到
- **补签**：`补🦌 5` 补签5号（只能补签没有签到过的日期）
- **戒🦌**：`戒🦌` 或 `戒🦌 5` 记录戒🦌（当天已签到则不能戒🦌）

#### 查询统计
- **#鹿鹿查询**：查看当月签到日历图片和统计数据（总次数、活跃天数、平均每天）
- **#鹿鹿查询上月**：查看上月签到日历图片和统计数据
- **#鹿鹿查询下月**：回复未来查询提示
- **#鹿鹿统计**：查看群内所有用户的统计和排行榜（仅群聊可用）
- **#总鹿鹿统计** / **#鹿鹿总统计**：查看全局统计和排行榜（所有群）

#### 健康管理
- **#鹿鹿体力查询**：查看当前精力值、健康度、恢复时间等
- **#鹿鹿健康报告**：查看详细的健康报告和健康建议

#### 数据管理
- **#清空鹿数据**：清空所有签到和体力数据（需要确认）
- **#强制清空鹿数据**：强制清空，无需确认
- **#确认清空鹿数据**：确认清空操作

## 精力系统说明

### 精力值系统
- **无上限**：精力值可以超过100，也可以为负数（透支）
- **基础消耗**：正常签到消耗25点，补签消耗30点
- **透支机制**：精力为负数时，消耗翻倍（一次=两次）
- **恢复机制**：每小时基础恢复2点，可加速

### 惩罚机制
- **连续天数惩罚**：每天增加15%消耗
- **当日次数惩罚**：每次增加30%消耗
- **肾脏健康度惩罚**：健康度越低，消耗越高
- **深夜惩罚**：23-5点额外50%消耗
- **透支惩罚**：透支时消耗翻倍

### 恢复机制
- **基础恢复**：每小时2点
- **戒🦌加成**：每天+10%恢复速度
- **睡眠加速**：22-6点恢复速度×2
- **戒🦌加速**：戒🦌期间恢复速度×1.5
- **健康度影响**：健康度<60时恢复速度×0.8

### 健康度系统
- **范围**：0-100
- **降低**：每次签到-1，补签-2
- **恢复**：戒🦌+1，长期戒🦌缓慢恢复
- **影响**：影响恢复速度和消耗

### 状态分级
- 🟢 **精力充沛**（80+）：肾气充足，精神饱满
- 🟡 **状态正常**（50-79）：精力尚可，注意节制
- 🟠 **精力不济**（20-49）：肾气亏损，需要休养
- 🔴 **精疲力尽**（0-19）：元气大伤，必须修养
- ⚫ **严重透支**（<0）：极度透支，强烈建议修养

## 系统特点

1. **无硬性限制**：不限制次数，但通过惩罚机制自然限制
2. **惩罚连续行为**：连续天数越多，消耗越大
3. **鼓励戒🦌**：戒🦌恢复精力并提升健康度
4. **时间机制**：深夜伤害更大，睡眠时段恢复更快
5. **长期健康**：健康度影响恢复和消耗
6. **可视化**：健康条显示当前状态

## 注意事项

- 精力透支后消耗会翻倍，建议及时修养
- 连续天数过多会大幅增加消耗
- 深夜（23-5点）消耗增加50%
- 健康度过低会影响恢复速度
- 戒🦌可以恢复精力和健康度
